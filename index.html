configurar ZSH y Oh-my-zsh en linux Ubuntu

Z Shell o zsh es un shell interactivo de UNIX y un potente intérprete de línea de comandos para el lenguaje de scripting, incluido el shell shell. Zsh ha sido desarrollado desde 1990 por 'Paul Fastad', y el apodo 'zsh' se deriva del profesor de Yale llamado Zong Shao.

El shell Z se ha convertido en uno de los shells más populares para el sistema operativo Linux. Es rico en características y fácil de configurar y personalizar. A continuación se presentan algunas características clave de zsh:

  • Comando auto-completado
  • Manejo mejorado de variables
  • Corrección ortográfica
  • Historial de comandos compartidos
  • Completar pestaña
  • Variable de entorno fácil de configurar

Shell de Unix

Una Shell de Unix o también shell, es el término usado en informática para referirse a un intérprete de comandos, el cual consiste en la interfaz de usuario tradicional de los sistemas operativos basados en Unix y similares como GNU/Linux.
